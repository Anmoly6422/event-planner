📅 Event Planner

A full-stack event management and RSVP platform built with Next.js App Router, Prisma ORM, and Neon Auth.
It enables users to create events, generate secure invite links, and collect RSVPs through a streamlined, server-driven architecture.

✨ Overview

Event Planner is designed to simplify event coordination by providing:

Event creation and management
Shareable invite links with secure tokens
RSVP collection system (Going / Maybe / Not Going)
Attendee tracking per event
Server-side authentication and authorization
Fully type-safe backend using Prisma

The application leverages Next.js Server Actions for seamless full-stack interactions without traditional REST APIs.

🚀 Features
🗓️ Create, view, and manage events
🔗 Unique invite link generation per event
🙋 RSVP system with real-time submission handling
👥 Attendee tracking with email normalization
🔐 Secure authentication using Neon Auth
⚡ Server Actions (no API routes required)
🧠 Prisma ORM with PostgreSQL database
📄 Dynamic routing with App Router
🧩 Modular UI using shadcn/ui components
📊 Event detail dashboard view
